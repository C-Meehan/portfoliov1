Candy Shop

This project was the result of a collaborative effort with fellow classmates from Coding Dojo. We created an E-commerce candy store utilizing React with a MongoDB Atlas database. I was responsible for front-end development alongside another classmate. Within this application a visitor to the site can browse a curated list of candy, filter through different categories and add items to their cart. I also created a secure administration login that allows the admin to add and alter products for the website. Github Live Demo

Venue Review

Venue Review is a stadium review website where users can create an account and review baseball stadiums they've visited. Curious how the food is at Citi Field? Wonder if you should sit in the outfield or behind home plate? With Venue Review you can check out the reviews from prior fans and leave your worries at home. Ratings are given on a 1-5 scale at .5 increments, and you can also leave a short-form review as well. Venue Review utilizes React with a MongoDB Atlas database. Github Live Demo

    Technologies used:

  • - React
  • - MongoDB
  • - HTML
  • - MaterialUI
